# 经典贪吃蛇

使用原生 HTML/CSS/JS 实现的最小版贪吃蛇（25x25 网格）。

## 运行

1. 在当前目录启动本地静态服务器：
   - `python3 -m http.server 5173`
2. 打开 [http://localhost:5173](http://localhost:5173)

## 测试

- 运行 `npm test`

## 手动验证清单

- 控制：
  - 方向键可控制移动。
  - `W`、`A`、`S`、`D` 也可控制移动。
  - `Space` 或“暂停”按钮可切换暂停/继续。
  - “自动游玩”按钮可切换自动路径选择。
  - 小屏/移动端下，屏幕方向按钮可用。
- 玩法：
  - 蛇按固定节拍在网格中移动。
  - 吃到食物后分数 +1，蛇身增长一格。
  - 食物不会生成在蛇身上。
- 结束/重开：
  - 撞墙后游戏结束。
  - 撞到自身后游戏结束。
  - “重新开始”会重置分数、蛇身与游戏状态。
